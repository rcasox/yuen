YUEN: Hakodate Yunokawa Project
지방 소멸 문제 해결을 위한 족욕바(Footbath Bar) 제안 웹사이트

프로젝트 개요
일정: 2026.02.05 - 2026.02.10

장소: 일본 하코다테 유노카와 온천 지역

협업: 하코다테 미래 대학(FUN) 학생 2명, 한국인 학생 2명

기획 배경
하코다테 유노카와 지역의 유휴 공간 및 온천 자원을 활용하여 젊은 층과 관광객의 유입을 유도하는 족욕과 바(Bar)가 결합된 공간 YUEN을 기획하고 이를 소개하는 웹 플랫폼을 구축했습니다.

참여 인원
Future University Hakodate: Matsunaga Koh, Okita Mahiru

Korea: 이예찬, 김민지

기술 스택 및 구조
Stack: HTML5, CSS3, JavaScript

Directory: html, css, js, image, video
